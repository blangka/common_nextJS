# common_nextJS

## 1. 프로젝트 목적
nextJS 를 사용하여 SSR , Recoil, typescript, react-query, css 등 다양한 기본이 될 프로젝트의 기본을 만듬  
creat-next-app 을 사용하게 되면 불필요한 파일들이 많이 생성되어서 개별적으로 설치 작업

## 2. 초기 환경 설정

### nextJS 기본 설치
```
npm i next react react-dom
```

### typescript 설정  
```
npm i -D typescript @types/react @types/node
```
